data/: Questa cartella contiene i dati utilizzati nel progetto.

raw/: Qui conservi i dataset originali non modificati.
processed/: Contiene i dataset elaborati e pronti per l'analisi e l'addestramento del modello.
notebooks/: Contiene i file Jupyter Notebook.

EDA.ipynb: Per l'analisi esplorativa dei dati. Puoi esplorare, visualizzare e analizzare i dati qui.
model_training.ipynb: Per l'addestramento del modello e la valutazione delle sue prestazioni.
src/: Contiene il codice sorgente del tuo progetto.

data_preprocessing.py: Script per caricare i dati e preprocessarli (pulizia, normalizzazione, codifica).
model.py: Contiene la definizione e l'addestramento del modello, inclusa la logica per la previsione.
evaluation.py: Script per valutare le performance del modello e visualizzare le metriche.
app.py: Script principale per avviare l'applicazione web (puoi scegliere di usare Flask o Streamlit).
requirements.txt: Questo file elenca tutte le dipendenze del tuo progetto. Puoi generarlo usando:
